Nova Motion Bot

Nova Motion Bot is a robot simulation project developed using Autodesk Fusion 360, ROS Noetic, and Gazebo. The robot model was designed in Fusion 360, exported for simulation, and integrated into a ROS 1 Gazebo environment for testing robot movement and behavior within a corridor-based scenario.

-Project Overview

The project focuses on creating a realistic simulated robot platform that can be used for testing navigation, motion control, and future sensor integration before deployment on physical hardware.

-Features

* Custom robot model designed in Autodesk Fusion 360
* ROS Noetic integration
* Gazebo simulation environment
* Corridor-based navigation testing
* Robot dynamics and motion simulation
* Modular structure for future sensor and control system integration

-Technologies Used

* ROS Noetic
* Gazebo
* Autodesk Fusion 360
* URDF/Xacro
* Python
* C++

-Simulation Scenario

The robot operates within a corridor environment where its movement, dynamics, and navigation behavior can be evaluated in a virtual setting. This allows testing and validation of robotic concepts before hardware implementation.

-Applications

* Robotics research and development
* Autonomous navigation testing
* Simulation-based validation
* Assistive robotics development
* Educational and academic projects

-Future Enhancements

* Camera integration
* Accelerometer and IMU integration
