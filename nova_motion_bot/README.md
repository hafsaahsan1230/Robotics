Nova Motion Bot

A ROS Noetic and Gazebo-based mobile robot simulation developed as part of an Assistive Robot project. The package provides a virtual robot environment for testing navigation, sensor integration, and autonomous robotic behaviors before deployment on physical hardware.

-Features

* Mobile robot simulation in Gazebo
* Robot model using URDF/Xacro
* ROS Noetic integration
* Launch files for simulation and visualization
* RViz support for robot monitoring
* Modular architecture for sensor and control integration

-Technologies

* ROS Noetic
* Gazebo
* RViz
* URDF/Xacro
* Python
* C++

-Applications

* Robot simulation and testing
* Sensor integration validation
* Navigation algorithm development
* Assistive robotics research

-Related Components

This simulation environment was used alongside additional ROS packages including camera and accelerometer integration modules within a larger assistive robot development project.
